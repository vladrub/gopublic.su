Конфигурация проекта

Создайте файл gopublic.yaml в корне проекта для запуска нескольких туннелей: 

version: "1"
tunnels:
{{- range $i, $d := .Domains}}
  tunnel-{{$i}}:
    proto: http
    addr: "{{add 3000 $i}}"
    subdomain: {{$d.Name}}{{end}}

Запуск клиента:

  • Базовый режим: gopublic start — читает gopublic.yaml из текущей директории и поднимает все туннели по конфигу.
  • Точка на один порт: gopublic start --port 8080 — без файла конфигурации, поднимает реверс-прокси только для указанного локального порта.

Запускать лучше из корня проекта, где лежит конфиг.

Условия использования

Для продолжения работы с сервисом необходимо принять условия использования.

При использовании GoPublic запрещается:

  • Распространение вредоносного ПО (вирусы, трояны, шпионские программы)
  • Фишинг и мошенничество (поддельные страницы для кражи данных)
  • Спам и массовые рассылки
  • Размещение нелегального контента
  • DDoS-атаки и обход блокировок

Нарушение данных правил влечёт немедленную блокировку аккаунта без предупреждения.

Полный текст условий доступен на странице Условия использования.
